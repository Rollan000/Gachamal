ℹ About GachaMAL
GachaMAL is a fan-made gacha browser game created by mal as a personal project for fun.

All anime character data, names, images, and related content are property of their respective copyright holders. This project uses the free and public Jikan API, which is an unofficial MyAnimeList API. GachaMAL is not affiliated with, endorsed by, or associated with MyAnimeList, any anime studios, publishers, or intellectual property owners in any way.

This project is:
Non-Commercial Fan-Made No Real Money Open Web / No Backend

No user data is collected or sent to any server. All game state is stored locally in your browser's localStorage. No authentication is required. Jikan API requests are made directly from your browser and are rate-limited by the API itself.

If you are a rights holder and have concerns about this project, please understand it is a non-profit fan creation with no monetization whatsoever. All characters are randomly pulled from public MAL data and used purely for play.
🎮 How It's Built
GachaMAL is a single-file HTML/CSS/JavaScript web app with no frameworks, build tools, or servers.

Character data — fetched in real time from api.jikan.moe (Jikan v4, random characters endpoint)
Stats — algorithmically derived from each character's MAL ID using deterministic pseudo-random seeding, so each character always has the same stats
Elements — assigned via a seeded hash of the MAL ID (Fire, Water, Wind, Lightning, Dark, Light, Earth, Ice)
Pity system — SR guaranteed every 10 pulls, SSR+ guaranteed at 90
Gear — randomized drops with set bonuses, attribute matching, and rarity tiers
Battle — turn-based speed bar system inspired by classic JRPGs and Etheria
Storage — all saved in your browser's localStorage under key "gm6"
